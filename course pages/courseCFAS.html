Computer Forensics and Security

What is this course about?

The BSc in Computer Forensics and Security at Canterbury Christ Church University prepares students for careers in cyber security amidst rising cybercrime. The program covers cyber security fundamentals, ethical hacking, and protection against system vulnerabilities. Students learn to use leading security tools like Autopsy, X-Ways, and Wireshark, and develop skills in civil litigation and criminal prosecution contexts. This degree emphasizes practical experience through collaboration and project management, ensuring graduates are well-equipped to handle complex security challenges in the workplace.
